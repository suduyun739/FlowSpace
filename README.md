# FlowSpace 🌌

您的个人工作流空间管理器

> 在FlowSpace中，每个工作流都有它的专属空间

## ✨ 功能亮点

### 🏠 个人工作空间
- 专属仪表板，一目了然
- 自定义工作区域
- 个性化设置

### 🎯 智能项目管理
- 看板视图拖拽操作
- 任务优先级管理
- 项目进度跟踪

### ✍️ 工作日记系统
- Markdown实时编辑
- 标签分类管理
- 全文搜索功能

### 📅 每日计划工具
- 智能日历选择
- 时间段任务规划
- 完成度统计

### 📊 数据可视化
- 工作统计图表
- 效率分析报告
- 趋势数据展示

## 🚀 快速开始

### 本地开发
```bash
# 克隆仓库
git clone https://github.com/suduyun739/FlowSpace.git
cd flowspace

# 启动本地服务器
python -m http.server 8000
# 或
npx serve .

# 访问应用
open http://localhost:8000
```

### 生产部署
```bash
# Docker构建
docker build -t flowspace .

# 运行容器
docker run -d -p 80:80 --name flowspace-app flowspace
```

## 🛠️ 技术栈

- **HTML5/CSS3** - 现代Web标准
- **Tailwind CSS** - 实用优先的CSS框架
- **JavaScript ES6+** - 原生现代JavaScript
- **Anime.js** - 流畅动画效果
- **ECharts.js** - 专业数据可视化
- **LocalStorage** - 安全本地存储

## 🎯 设计理念

FlowSpace秉承"简约而不简单"的设计哲学：
- **极简界面** - 减少视觉干扰，专注工作内容
- **直观交互** - 符合直觉的操作逻辑
- **响应式设计** - 完美适配各种设备
- **深色主题** - 护眼且专业的外观
