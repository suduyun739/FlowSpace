## 🚀 快速开始

### 本地运行
```bash
# 进入项目目录
cd /mnt/okcomputer/output

# 启动本地服务器
python -m http.server 8000

# 访问应用
http://localhost:8000
```

### Docker部署
```bash
# 构建镜像
docker build -t flowspace .

# 运行容器  
docker run -d -p 80:80 flowspace
```

### GitHub推送
```bash
# 运行推送脚本
./push-to-github.sh

# 或手动推送
./push-to-github.sh
```
